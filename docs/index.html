Clinigest

Sistema web para gestão de clínica psicológica, desenvolvido com Flask, SQLite, HTML, CSS e JavaScript.

Ver repositório Como executar

Sobre o projeto

O Clinigest foi criado para facilitar a organização de pacientes, consultas, sessões e movimentações financeiras de uma clínica psicológica. A proposta é centralizar informações importantes em uma aplicação simples, organizada e acessível localmente.

Principais funcionalidades

Pacientes

Cadastro, edição, prioridade e relatório individual.

Agenda

Agendamento, edição e acompanhamento de consultas.

Sessões

Registro de observações e histórico de atendimentos realizados.

Financeiro

Controle de entradas, saídas e saldo geral da clínica.

Calendário

Visualização das consultas pendentes em formato de calendário.

Dashboard

Resumo com indicadores importantes do sistema.

Diferenciais técnicos

  • Aplicação organizada em rotas Flask.
  • Uso de banco de dados SQLite com SQLAlchemy.
  • Sistema de login para restringir o acesso.
  • Templates HTML reutilizáveis com estrutura base.
  • Integração entre consultas, pacientes e financeiro.

Tecnologias utilizadas

Como executar a aplicação

O GitHub Pages hospeda apenas a página de apresentação. Para rodar o sistema Flask, execute localmente:

git clone https://github.com/natiwelter/clinigest.git
cd clinigest
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py

Depois, acesse:

http://127.0.0.1:5000
